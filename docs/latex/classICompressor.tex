\doxysubsection{Detailed Description}
Abstract interface class for compression and decompression algorithms. 

This interface defines the required methods for any compression class. Classes inheriting from \mbox{\hyperlink{classICompressor}{ICompressor}} must implement both compression and decompression functions.
